Customizable Dash App

December, 2016 by eyobw

Customizable-Dash is a browser based client side dashboard feamework. The framework takes a dataset in JSON format and lets the user to design the dashboard, with certain flexibility such as, the user could draw frequency bar and pie charts, bar charts to compare prperties of datasets and line chart for sequencial data depending on the property of the dataset provided by the user.

Upload your JSON data and from the side navigation bar click on the graph type you would like to draw

SampleLine Chart form looks like this, optionally use the filtering parameter to filter the data, and choose X and Y values for the chart. You can select as many y parameters as you like.

Npm Install

If you wish to install the application on your computer, you need to have npm, grunt and bower installed on your computer. Then install all dependencies using the following command on terminal.

grunt build
grunt serve

The application should start on your default browser at http://localhost:9000/#/
